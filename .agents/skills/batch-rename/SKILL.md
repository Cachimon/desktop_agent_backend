# Instructions
## 功能
支持对单个或多个文件进行批量重命名，重命名的方式支持多样，不同方式以不同的脚本实现。
本skill已配套好实现的脚本，可根据用户的诉求执行对应的脚本。
但如果需要执行脚本，则麻烦通过 `execute_skill_script` 的工具调用方式来执行，传入本skill的名称、脚本的文件名称及参数。

## 执行流程
1. 用户需要提供目标目录的路径
2. 通过执行命令 `dir /b /ad <目标路径>` 查看是否有文件夹，如果没有文件夹，则执行步骤3，跳过步骤4。如果有，则需要向用户确认是否需要递归处理，如果需要递归，则跳过步骤3，直接执行步骤4；如果不需要递归，则执行步骤3，跳过步骤4。
3. 获取目录路径下的非递归的文件路径列表
4. 获取目录路径下递归的文件路径列表
5. 根据用户的重命名规则，通过大模型生成对应新的文件路径列表，并生成预览结果
6. 通过 `ask_user` 工具将预览结果返回给用户确认
7. 如果用户确认继续执行，则通过 `execute_skill_script` 工具执行脚本 `rename.py`，脚本参数如下方参数列表所示，请严格使用以下字段名，传给 `execute_skill_script` 工具的args参数；如果用户取消，则结束流程

## 脚本列表
### `rename.py`

#### Parameters
请严格使用以下字段名：

- `old_file_list`: List[str] - 原始文件路径列表
- `new_file_list`: List[str] - 目标文件路径列表

示例参数：
```json
{
  "old_file_list": ["1.png", "2.md", "3.py"],
  "new_file_list": ["hello_1.png", "hello_2.md", "hello_3.py"]
}
```


#### 返回格式
- 成功时：返回0
- 失败时：返回错误消息，说明失败原因（如文件未找到、格式错误等）
